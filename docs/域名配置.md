# 域名配置指南

## 📋 域名信息

| 平台 | 域名 | 用途 | 对应项目 |
|------|------|------|----------|
| **API 托管站** | `freeapi.cc.cd` | Provider 端 - AI 账号托管、收益管理 | `apps/provider` |
| **API 购买站** | `freecode.cc.cd` | Consumer 端 - API 购买、客户密钥、返佣 | `apps/consumer` |

---

## 🔧 Cloudflare DNS 配置

### 步骤 1：添加域名到 Cloudflare

1. 登录 Cloudflare Dashboard
2. Add a Site → 输入 `cc.cd`
3. 选择 Free Plan
4. 更新 Nameservers（到域名注册商处修改）

### 步骤 2：添加 DNS 记录

#### API 托管站 (freeapi.cc.cd)

```
类型：CNAME
名称：freeapi
内容：<cloudflare-pages-subdomain>.pages.dev
代理状态：Proxied (橙色云朵)
```

#### API 购买站 (freecode.cc.cd)

```
类型：CNAME
名称：freecode
内容：<cloudflare-pages-subdomain>.pages.dev
代理状态：Proxied (橙色云朵)
```

### 步骤 3：SSL/TLS 设置

```
加密模式：Full (完全)
始终使用 HTTPS：启用
自动 HTTPS 重写：启用
```

## 🚀 Cloudflare Pages 部署

### 创建项目

#### 项目 1：API 托管站

1. Pages → Create a project
2. Connect to Git → 选择 `derouter-clone`
3. 配置：
   - **Project name**: `freeapi`
   - **Production branch**: `main`
   - **Root directory**: `apps/provider`
   - **Build command**: `npm install && npm run build`
   - **Build output directory**: `.next`

#### 项目 2：API 购买站

1. Pages → Create a project (或 Add another project)
2. 选择同一个仓库
3. 配置：
   - **Project name**: `freecode`
   - **Root directory**: `apps/consumer`
   - **Build command**: `npm install && npm run build`
   - **Build output directory**: `.next`

### 绑定自定义域名

#### freeapi.cc.cd

1. freeapi 项目 → Custom domains
2. Add custom domain
3. 输入：`freeapi.cc.cd`
4. 自动添加 DNS 记录（如未手动添加）
5. 等待 SSL 证书颁发（约 5-10 分钟）

#### freecode.cc.cd

1. freecode 项目 → Custom domains
2. Add custom domain
3. 输入：`freecode.cc.cd`
4. 同上

## 📝 环境变量配置

### API 托管站 (freeapi.cc.cd)

在 Cloudflare Pages 项目设置中添加：

```env
# Privy (Web3 登录)
NEXT_PUBLIC_PRIVY_APP_ID=privy_app_id_here
PRIVY_APP_SECRET=privy_app_secret_here

# Solana
NEXT_PUBLIC_SOLANA_RPC_URL=https://api.mainnet-beta.solana.com

# 项目标识
NEXT_PUBLIC_PLATFORM=freeapi
```

### API 购买站 (freecode.cc.cd)

```env
# Supabase
NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=anon_key_here
SUPABASE_SERVICE_ROLE_KEY=service_role_key_here

# 项目标识
NEXT_PUBLIC_PLATFORM=freecode
```

## 🔐 SSL 证书

Cloudflare 自动提供 SSL 证书：

- **类型**：Universal SSL
- **有效期**：自动续期
- **覆盖**：`*.cc.cd`

### 验证 SSL

```bash
# 检查 SSL 证书
curl -I https://freeapi.cc.cd
curl -I https://freecode.cc.cd

# 检查 HTTPS 重定向
curl -I http://freeapi.cc.cd  # 应 301 到 HTTPS
curl -I http://freecode.cc.cd  # 应 301 到 HTTPS
```

## ⚠️ 注意事项

1. **DNS 传播**：修改后可能需要 24-48 小时全球生效
