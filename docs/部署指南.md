# 部署指南

## 📦 部署到 Cloudflare Pages

### 前置准备
1. Cloudflare 账号
2. 域名（已添加到 Cloudflare DNS）
3. GitHub 仓库

### 步骤

#### 1. 推送代码到 GitHub
```bash
cd /Users/gang/.openclaw/workspace/projects/derouter-clone
git init
git add .
git commit -m "Initial commit - DeRouter clone project"
git branch -M main
git remote add origin <your-github-repo-url>
git push -u origin main
```

#### 2. Cloudflare Pages 部署

**托管平台 (Provider)**
1. 登录 Cloudflare Dashboard
2. 进入 Pages → Create a project
3. Connect to Git → 选择 derouter-clone 仓库
4. 配置：
   - **Project name**: `derouter-provider`
   - **Production branch**: `main`
   - **Build command**: `npm install && npm run build --workspace=apps/provider`
   - **Build output directory**: `apps/provider/.next`
   - **Root directory**: `apps/provider`
5. Deploy

**采购平台 (Consumer)**
1. 同上，创建新项目
2. 配置：
   - **Project name**: `derouter-consumer`
   - **Build command**: `npm install && npm run build --workspace=apps/consumer`
   - **Build output directory**: `apps/consumer/.next`
   - **Root directory**: `apps/consumer`

#### 3. 自定义域名
1. Pages 项目 → Custom domains
2. 添加域名：
   - Provider: `provider.yourdomain.com` 或 `yourdomain.network`
   - Consumer: `api.yourdomain.com` 或 `yourdomain.ai`

## 📦 部署到 Vercel

### 步骤

#### 1. 推送代码到 GitHub（同上）

#### 2. Vercel 部署

**托管平台 (Provider)**
1. 登录 Vercel → Add New → Project
2. Import Git Repository → 选择 derouter-clone
3. 配置：
   - **Project Name**: `derouter-provider`
   - **Root Directory**: `apps/provider`
   - **Build Command**: `npm run build`
   - **Output Directory**: `.next`
4. Deploy

**采购平台 (Consumer)**
1. 同上，创建新项目
2. 配置：
   - **Project Name**: `derouter-consumer`
   - **Root Directory**: `apps/consumer`

#### 3. 自定义域名
1. Project Settings → Domains
2. 添加域名

## 🔧 环境变量配置

### 托管平台 (.env.local)
```env
# Privy
NEXT_PUBLIC_PRIVY_APP_ID=your_privy_app_id

# Solana
NEXT_PUBLIC_SOLANA_RPC_URL=https://api.mainnet-beta.solana.com

# 后端 API
NEXT_PUBLIC_API_URL=https://api.yourdomain.com
```

### 采购平台 (.env.local)
```env
# Supabase
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key

# 后端 API
NEXT_PUBLIC_API_URL=https://api.yourdomain.com
```
