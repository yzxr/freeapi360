## 🔐 登录系统

### 登录流程
```
1. 输入邮箱 → 2. 发送验证码 → 3. 6 位数字验证 → 4. 自动登录
```

### 技术特点
- **无密码设计**：纯邮箱 + 验证码登录
- **验证码时效**：36 秒后可重发
- **会话保持**：登录后保持登录状态
- **退出机制**：导航栏右上角显示邮箱 + 退出按钮

### 安全机制
- 服务条款同意勾选（登录即表示同意）
- 隐私政策链接
- 客服咨询入口（Telegram）

## 💰 商业模式拆解

### 收入来源

| 来源 | 说明 | 利润率 |
|------|------|--------|
| **API 调用差价** | 托管账号成本 vs 向用户收费 | 50-70% |
| **平台抽成** | Provider 收益的 25% | 纯利润 |
| **资金沉淀** | 用户预充值余额 | 现金流 |

### 成本结构

| 成本项 | 说明 | 占比 |
|--------|------|------|
| AI 账号订阅 | Claude Pro/OpenAI 订阅费 | ~60% |
| 服务器/带宽 | API 网关、数据库、CDN | ~20% |
| 链上结算 | Solana 交易费用 | ~5% |
| 客服/运营 | 人力成本 | ~15% |

### 分销体系

```
推荐人 (30% 返佣) ← 平台 (25% 抽成) ← Provider (75% 收益)
```

## ⚠️ 风险点分析

### 1. 账号共享合规性
- **风险**：违反 Claude/OpenAI 服务条款
- **后果**：账号封禁、法律风险
- **缓解**：限制单账号请求频率、模拟真实用户行为

### 2. 技术风控
- **风险**：AI 平台检测异常使用模式
- **对策**：
  - IP 轮换
  - 请求间隔随机化
  - User-Agent 轮换
  - 会话管理

### 3. 资金安全
- **风险**：加密货币波动、洗钱嫌疑
- **对策**：
  - 法币通道（微信支付）
  - KYC 验证（大额提现）
  - 交易限额

### 4. 数据隐私
- **风险**：用户对话内容泄露
- **对策**：
  - 端到端加密
  - 不存储对话内容
  - 隐私政策明确
